# Signal Strength Prediction Using Neural Networks and Deep Learning

## Overview

This project was developed as part of a Neural Networks and Deep Learning (NN & DL) course. The objective is to build and train an Artificial Neural Network (ANN) model to predict signal strength using network-related features and measurements.

The project demonstrates the end-to-end deep learning workflow, including data preprocessing, exploratory data analysis, neural network design, model training, validation, and performance evaluation.

## Domain

Telecommunications / Wireless Networks

## Problem Statement

Signal strength prediction plays an important role in network optimization, coverage planning, and communication reliability. This project leverages deep learning techniques to model complex relationships between network parameters and signal strength.
